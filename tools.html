Research Software

Mission intelligence tools

ORACLE-MESH provides interoperable modules for probabilistic reasoning, graph-native data assimilation, and policy synthesis in complex dynamical missions.

T1

Networked Gaussian Processes

Distributed GP blocks over mission graphs for spatial, temporal, and relational uncertainty inference.

T2

GP-FSM Engine

Gaussian Process Finite State Machine layer for state transitions with probabilistic guards and uncertainty-aware mode switching.

T3

Tactical Options Trees

Hierarchical options tree generation with branch confidence scoring for mission planning and contingency analysis.

T4

Graph Uncertainty Propagation

Message-passing operators for uncertainty transport across graph edges, latent factors, and constraints.

T5

Data Assimilation on Graphs

Sequential and batch assimilation primitives for updating graph states from heterogeneous sensing streams.

T6

RL-guided Mission Policies

Policy synthesis with uncertainty-aware reward shaping and risk controls for adaptive mission behavior.

Integration model

Composable modules, shared uncertainty state.

Each tool publishes and consumes a common uncertainty state representation. This keeps model updates, finite-state transitions, graph diffusion, and policy control aligned during both offline mission rehearsal and online execution.
