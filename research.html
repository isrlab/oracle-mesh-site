Research Program

This page summarizes the motivating imperatives, core technical architecture, and primary research thrusts enabling real-time, edge-deployed AI for uncertainty-aware decision support in aerospace autonomous systems.

Research Product

D3ARWIN

Dynamic Data-Driven Adaptive Reasoning With Inference Networks

The project is motivated by the need for dependable, distributed decision intelligence when sensing, models, and compute are all imperfect. It organizes uncertainty-aware inference and coordination into a unified workflow so edge systems can maintain reliable performance under shift, partial observability, and communication constraints.

In practice, the framework couples evidence assimilation, belief coordination, and robust action selection into a single loop: combine uncertain observations, reconcile local and shared model context, and output decisions with explicit confidence and safety margin awareness, producing certifiable autonomy grounded in quantified uncertainty rather than black-box prediction.

Core Technical Innovations

Six interlocking technical advances form the computational foundation of D³ARWIN. Each is independently novel; together they constitute a closed-loop architecture for uncertainty-aware decision intelligence at the edge.

hub

Situational Awareness

GP-Finite State Machine

The mission options space is represented as a finite automaton where each node is a Gaussian process. Sensor observations drive posterior state estimation across the graph, producing principled situational awareness in partially observable, dynamically evolving environments.

schema

Physics-Informed Surrogates

Networked Deep Gaussian Processes

A hierarchical network of physics-informed GP components models coupled multi-physics domains: aerodynamics, structures, propulsion, and flight dynamics. Coupling interfaces propagate calibrated uncertainty through the network topology without requiring separate coupling models.

sync_alt

Real-Time State Estimation

Data Assimilation on Graphs

Graph signal processing integrates streaming sensor data across GP networks. Uncertainty propagates through graph diffusion operators governed by network topology, while Bayesian information fusion continuously updates posterior state distributions as new observations arrive.

tune

Adaptive Runtime Fidelity

Dialable-Fidelity Digital Twin

Certified link-reduction approximations allow the surrogate network to transition between high-fidelity offline analysis and low-latency onboard prediction. Each reduced mode is validated against the full reference model with conservative uncertainty adjustments, preserving safety margin consistency across fidelity tiers.

query_stats

Uncertainty Quantification

Bayesian-Agentic UQ Framework

Model refinement is formulated as a sequential decision problem. A computational agent allocates fidelity by optimizing over prediction error, confidence bounds, and computational cost, distinguishing epistemic from aleatoric uncertainty and directing refinement only where it produces operationally meaningful improvement.

psychology

Policy Optimization

Deep Reinforcement Learning on Graphs

A DRL agent learns decision policies within the GP-FSM framework. GP posteriors provide risk-aware reward signals and probabilistic action evaluation, enabling continuous policy refinement across mission scenarios with quantified safety margins and explicit uncertainty budgets.
