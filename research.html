Research Program

This page summarizes the motivating imperatives, core technical architecture, and primary research thrusts enabling real-time, edge-deployed AI for uncertainty-aware decision support in aerospace autonomous systems.

Research Product

D3ARWIN

Dynamic Data-Driven Adaptive Reasoning With Inference Networks

The project is motivated by the need for dependable, distributed decision intelligence when sensing, models, and compute are all imperfect. It organizes uncertainty-aware inference and coordination into a unified workflow so edge systems can maintain reliable performance under shift, partial observability, and communication constraints.

In practice, the framework couples evidence assimilation, belief coordination, and robust action selection into a single loop: combine uncertain observations, reconcile local and shared model context, and output decisions with explicit confidence and safety margin awareness, producing certifiable autonomy grounded in quantified uncertainty rather than black-box prediction.

Core Technical Innovations

Six interlocking advances form the computational backbone of D³ARWIN. Each solves a distinct operational bottleneck; together they close the loop from raw sensor data to certifiable, uncertainty-aware decisions at the edge.

hub

Situational Awareness

GP-Finite State Machine

Maps the full mission option space as a graph of Gaussian processes. As sensor data streams in, the system tracks which operational state the vehicle is in, what it can transition to, and how confident it should be, even when the environment is only partially observable.

schema

Physics-Informed Surrogates

Networked Deep Gaussian Processes

Chains physics-grounded GP components (aero, structures, propulsion, flight dynamics) into a single surrogate network. Uncertainty flows through coupling interfaces automatically, so every downstream prediction carries a calibrated confidence bound without bolting on a separate error model.

sync_alt

Real-Time State Estimation

Data Assimilation on Graphs

Performs Bayesian fusion over the graph-structured surrogate network, conditioning GP posteriors on live sensor data and propagating updated beliefs across nodes. Graph signal processing methods accelerate the propagation step, avoiding full matrix inversions at scale while preserving calibrated uncertainty throughout the network.

tune

Adaptive Runtime Fidelity

Dialable-Fidelity Digital Twin

Lets the system dial between high-fidelity offline analysis and low-latency onboard prediction through certified link-reduction approximations. Each reduced mode is validated against the full reference model with conservative uncertainty adjustments, preserving a traceable fidelity chain for certification.

query_stats

Uncertainty Quantification

Bayesian-Agentic UQ Framework

Treats model refinement as a resource allocation problem. An autonomous agent decides where to spend compute by weighing prediction error against confidence bounds and latency, distinguishing what the model does not know from what is inherently noisy, and refining only where it moves the needle operationally.

psychology

Guidance & Decision Synthesis

Uncertainty-Aware Planning & Control

Computes guidance and control actions directly on the GP-FSM graph in real time. The framework is solver-agnostic: reinforcement learning, model-predictive control, or any successive approximation method can plug into the same uncertainty-rich graph structure. As posteriors update, the planner re-evaluates candidate actions against current confidence bounds, delivering low-latency decisions with quantified safety margins even as the operational picture shifts mid-mission.
