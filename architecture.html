Portal Blueprint

Website architecture and stack

This page defines the initial ORACLE-MESH research software portal architecture for GitHub Pages deployment.

Website architecture

Information architecture

  • Home: mission statement, capability framing, and navigation entry points
  • Tools: research module catalog and integration contracts
  • Visualizations: interactive demos for graph diffusion, GP-FSM, and options trees
  • Docs: onboarding, API structure, and operator guides
  • Architecture: system blueprint and implementation stack
  • About: team profiles and technical leadership

Technology stack

Initial implementation stack

  • Hosting and deployment: GitHub Pages (static site from main branch)
  • Frontend: semantic HTML5 with responsive CSS design system
  • Interaction layer: vanilla JavaScript for canvas and DOM-driven demos
  • Visualization primitives: HTML canvas for graph and finite-state animations
  • Content model: page-level modular sections for tools, docs, and architecture

Initial pages

Portal launch pages
